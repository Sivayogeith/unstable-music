# Unstable Music 
This resource pack adds (and **removes default minecraft music**) popular Unstable Universe music to the game!
>[!TIP]
>I suggest you to use [Simple Music Control](https://modrinth.com/mod/simple-music-control) mod with this resource pack!

## Song List
- AZALI - Battle Under A Broken Sky
- AZALI - Chaos Construct
- Jon Presstone - Extrastellar
- Kyon - Sorcery Ring
- Nombra101 - FlameFrags' 2nd Official Theme
- Nombra101 - FlameFrags' Official Theme
- Nombra101 - Scorched Earth Symphony
- Nombra101 - Oblivionfire
- NourOSTs - It's Fraggin Time
- NourOSTs - Last Chance

If you want me to add a track you want, make an issue in my github repo!
